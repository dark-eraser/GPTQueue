## Features

- Send multiple prompts simultaneously
- Support for multiple LLMs

## Requirements
`npm`
## Installation

Install GPTQueue with npm

```bash
    git clone git@github.com:dark-eraser/GPTQueue.git && cd GPTQueue
    npm install .
```
    
## Deployment

To deploy this project run:

```bash
  npm run deploy
```
